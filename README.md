# Delicious Eats Restaurant Website

## Description

This project is a responsive multi-page restaurant website built with HTML and CSS. The website includes a home page, menu page, and contact page.

## Improvements Made

* Added responsive layouts using Flexbox and media queries
* Added responsive images using srcset and sizes attributes
* Added skip navigation links for accessibility
* Added keyboard focus indicators for navigation
* Improved semantic HTML structure
* Added a dedicated Contact page
* Added viewport meta tags for mobile responsiveness
* Validated HTML and CSS code

## Tools Used

* Visual Studio Code
* Chrome DevTools
* W3C HTML Validator
* W3C CSS Validator
* Git and GitHub

## How to View the Website

Open index.html in a modern web browser and use the navigation menu to access the Menu and Contact pages.
